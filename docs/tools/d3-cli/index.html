沙皇的工具

简介

说到数据可视化，就不能不谈到 Mike Bostock。他创作了最有名的 d3 。 他最新的创新就是 d3-geo 的命令行。在 echarts 的地图编辑工作里，很多技术的问题需要他的工具。

安装

  1. 首先，需要 node.js。我在其他章节里提到过如何安装 node.js，这里就省略了。

  2. 再需要就是用 npm

    npm install -g d3-geo-projection

主要功能

geoproject

geoproject 对 geojson 进行旋转，位移，和投影。
